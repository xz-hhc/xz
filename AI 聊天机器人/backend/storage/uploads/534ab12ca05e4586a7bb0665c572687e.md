# AI 聊天机器人知识库

## 项目介绍
AI 聊天机器人是一个基于 RAG（检索增强生成）技术的智能问答系统。
用户可以通过上传文档资料来训练自己的AI机器人，实现基于个人知识库的智能问答。
系统支持 TXT、PDF、DOCX、CSV、MD、JSON 等常见文档格式。

## 核心功能
1. 文档上传与处理：用户上传文档后，系统会自动提取文本内容，进行中文分词处理
2. 知识库构建：将文档切分为知识片段，通过 TF-IDF 向量化技术建立索引
3. 智能问答：用户提问时，系统检索相关知识片段，生成有针对性的回答
4. 知识库管理：用户可以查看已上传的文档列表，删除不需要的文档

## 技术架构
后端使用 Python FastAPI 框架，提供高性能的异步 API 服务。
文本向量化采用 TF-IDF 算法，结合 jieba 中文分词，实现精确的语义检索。
TF-IDF 通过统计词频和逆文档频率来衡量词语的重要性。
余弦相似度用于计算用户问题和知识片段之间的相关度。

## 数据流程
1. 用户上传文档 → 系统提取文本内容
2. 文本切分为知识块 → 每个块约500字符，块之间有100字符重叠
3. 中文分词处理 → 提高检索精度
4. TF-IDF向量化 → 建立向量索引
5. 用户提问 → 问题向量化 → 相似度检索 → 生成回答

## 系统优势
- 无需GPU即可运行：使用传统的TF-IDF算法，CPU上即可高效运行
- 中文优化：使用jieba分词器进行中文分词，准确识别中文语义
- 轻量部署：后端总依赖小于100MB，适合云服务器部署
- 灵活扩展：支持多种文件格式，可接入微信小程序等多种前端
